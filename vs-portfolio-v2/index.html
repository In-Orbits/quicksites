Product Reference 7 products

VergeSense builds ceiling-mounted workplace occupancy sensors and IoT gateways for enterprise space management.

L302 Sensor · Battery

Last-gen battery-powered ceiling occupancy sensor. PIR sensor triggers wake event, camera takes a picture, YOLO model runs wireframe over image, AI/ML model detects people and objects. Transmits via Wirepas mesh to gateway to cloud.

AI: Kendryte K210 (Canaan) running YOLO object detection
Wireless: u-blox BMD340 (Nordic nRF52-based) running Wirepas mesh
Power: L91 Battery-powered
Camera: Carlco, CIF resolution (352×288)
Mechanical: Magnet, snap-on bracket, drop rod
L410 Sensor · Battery · Next-Gen

Next-gen battery-powered ceiling occupancy sensor, replacing L302. Similar signal pipeline and networking workflow with hardware upgrades throughout.

AI/Compute: Alif E3 (22nm SOI, GlobalFoundries)
Wireless: Nordic nRF52840 running Wirepas mesh
Power: Custom lithium thionyl chloride pack — 3× ER14505 bobbin cells + SPC0920A supercap. Targeting 10-year battery life.
Camera: Bison module, Hynix image sensor, Commonlands lens. 120° FOV.
Mechanical: Magnet, snap-on bracket, drop rod. Injection-molded paintable skins.
E106 Sensor · PoE · Wired

PoE-powered wired ceiling occupancy sensor. No PIR needed — continuous PoE power enables higher polling rate and better camera. Supports both Wirepas mesh and 802.11 networking.

Compute: Variscite VAR-SOM-MX8M-Mini (NXP i.MX8M Mini) — running YOLO object detection
Camera: Bison camera module
Wireless: Panasonic PAN1770 (Nordic nRF52840) running Wirepas mesh
Power: PoE input (48–56V, 12.95W Class 2) via MPS MP8017
Connectivity: USB-C & Ethernet
Mechanical: Magnet, snap-on bracket, drop rod
G400-C Gateway · Cellular

IoT gateway with cellular connectivity, replacing Rigado Cell. Connects to cellular network, runs Wirepas mesh to bridge L410/L302 sensors to cloud. Heavy derivative of E106 platform.

Compute: Variscite VAR-SOM-MX8M-Mini (same as E106)
Cellular: Quectel EG25-GL CAT4 modem. 2× external antennas. Internal SIM (AT&T default).
Wireless: Panasonic PAN1770 (Nordic nRF52840) running Wirepas mesh
Power: USB-C (5V @ 3A) or PoE via MPS MP8017
Mechanical: Magnet, snap-on bracket, drop rod, wall mount, tombstone
G400-NC Gateway · Non-Cellular

IoT gateway, Ethernet/PoE/802.11 only. Depopulated G400-C — no cell modem, no external antennas. Same Wirepas mesh gateway function over wired network.

Rigado Cell 3rd Party · EOL

Third-party IoT gateway (cellular + non-cellular variants). Supported Wirepas mesh bridging. VergeSense paid per-unit license fee. EOL'd and replaced by G400 to eliminate licensing dependency. Rigado acquired by Walmart to deploy IoT gateways across their stores and warehouses.

Xovis EN1 3rd Party · Resold

Third-party stereoscopic people counter resold by VergeSense. PoE-powered, ceiling-mounted above doorways. High-frequency, high-accuracy counting for high-traffic entry points (airports, lobbies, building entries). Swiss-designed, premium cost. Integrated into VergeSense platform for occupancy analytics — complements L302/L410/E106 room-level sensors with directional threshold counting.
